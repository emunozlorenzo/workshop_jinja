2. Data Scope

The time series modelized is Timelab Model - ARIMA Var0. The data chosen for the model’s development has been filtered from 2011-12-31 00:00:00 to 2020-03-31 00:00:00. The time series are on a monthly data. The dates, which have been taken into account, are as follows:

  • Start train: 2011-12-31 00:00:00
  • Start test: 2017-03-31 00:00:00
  • Start forecast: 2020-03-31 00:00:00
  • End forecast: 2025-12-31 00:00:00
3. Data Outliers and Smoothing

The transformations applied to the series are the following (listed in order of application):

  1. Outliers:

    The ouliers detected have been interpolated.
    • var0: No outliers.
    • var1: No outliers.

  2. Smoothing:

    Filter used to smooth the data in order to reduce the excess in volatility
    • var0: Hodrick-Prescott filter; Smoothing Factor 1.0.
    • var1: Hodrick-Prescott filter; Smoothing Factor 1.0.

  3. Differencing:

    • var0: Relative Differences. Where p= 1 stands for periods and refers to the window in which the transformation has been done.
    • var1: Relative Differences. Where p= 1 stands for periods and refers to the window in which the transformation has been done.

  4. Lags:

    • var0: 0
    • var1: 0
The resulting time series after the former transformations has achieved a stationarity score on Augmented Dickey-Fuller Test of: p-value=0.003
